Вариант 2

Размерности массивов задаются именованными константами. Дан  массив, состоящий из n  вещественных чисел. 
    1. Найти наибольший и наименьший элементы массива и их индексы.
    2. Поменять местами наименьший элемент массива с первым элементом, а наибольший с последним элементом.
    3. Выполнить сортировку массива по убыванию.
